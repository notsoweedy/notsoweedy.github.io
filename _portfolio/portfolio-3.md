## Overview

Developed during an internship at National University of Singapore, this Heart Stroke Prediction Model uses machine learning to predict the likelihood of heart strokes based on specific healthcare and lifestyle factors. The model achieved an impressive accuracy of 96.8%.

## Key Features

- **High Accuracy**: Achieved 96.8% prediction accuracy
- **Healthcare Analytics**: Analyzes specific healthcare and lifestyle factors
- **Multiple Algorithms**: Tested various architectures including SVM, Naive Bayes, Random Forest, and Logistic Regression
- **Performance Analysis**: Comprehensive comparison of different ML algorithms
- **Data-Driven Insights**: Provides insights into key risk factors for heart strokes

## Research & Analysis

Analyzed the performance of various machine learning architectures to determine the most effective approach:
- Support Vector Machines (SVM)
- Naive Bayes
- Random Forest
- Logistic Regression

The comparative analysis helped identify the optimal model for heart stroke prediction.

## Impact

This model can help healthcare professionals identify at-risk patients early and implement preventive measures, potentially saving lives through early intervention.
